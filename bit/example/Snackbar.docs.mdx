## Feedback / Snackbar

Snackbars provide brief messages about app processes.  The component is also known as a toast.

### Component usage
```js
<Snackbar text="hello from Snackbar" />
```

### Customization

Modify the text to see it change live:
```jsx live
<Snackbar text="hello from Snackbar" />
```
