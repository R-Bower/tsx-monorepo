# System Props

Our React components utilize [styled-system](https://github.com/jxnblk/styled-system)'s style functions to apply groups of props to each component.   These functions modify our components to accept CSS variables as props.

Many components get the `COMMON` set of props which give the component access to background, border, color, sizing, space, and transition styling.  Refer to the [table](/system-props#style-props-table) below for each value.

## Colors
Colors are overridden and restricted to an interface.
