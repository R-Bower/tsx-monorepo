# Grid

Grid is a component that exposes grid system props. Learn more about Grid Layout [here](https://gridbyexample.com/learn/).

## Default example

```jsx live
<Grid gridGap={3} gridTemplateColumns={"repeat(2, auto)"}>
  <Box bg={"bg.infoInverse"} color={"text.inverse"} p={3}>
    1
  </Box>
  <Box bg={"bg.warningInverse"} color={"text.inverse"} p={3}>
    2
  </Box>
</Grid>
```

## Component props

| Prop name | Type   | Default | Description                         |
| :-------- | :----- | :-----: | :---------------------------------- |
| as        | String |  `div`  | sets the HTML tag for the component |
