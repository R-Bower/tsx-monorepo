# Text

The Text component is used to apply typography styles to text.

```jsx live
<Text fontWeight={"bold"} variant={"p"}>bold</Text>
<Text color={"text.danger"} variant={"p"}>danger color</Text>
<Text cursor={"pointer"} variant={"p"}>pointer cursor</Text>
```

## Component props

| Name      | Type        | Default | Description                                |
| :---      | :-----      | :-----: | :-----------------------------------------:|
| as        | String      | `null`  | Sets the HTML tag for the component        |
| variant   | String      | `span`  | Sets the style variant for the component   |

## System props

Text components get `TYPOGRAPHY`, `INTERACTIVITY`, and `COMMON` system props.
