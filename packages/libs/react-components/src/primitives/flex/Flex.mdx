# Flex

The `Flex` component is a superset of the [Box](/doc/components/primitives/box) component with the following additional traits:
- it has `display: flex` set by default.
- it has access to the `FLEX` category of system props.

```jsx live
<Flex
  border={"solid 1px"}
  borderColor={"border.primary"}
  borderRadius={2}
  height={300}
  width={1}
>
  <Flex flexWrap={"nowrap"}>
    <Flex bg={"bg.infoInverse"} color={"text.inverse"} p={3}>
      Item 1
    </Flex>
    <Box bg={"bg.warningInverse"} color={"text.inverse"} p={3}>
      Item 2
    </Box>
    <Box bg={"bg.dangerInverse"} color={"text.inverse"} p={3}>
      Item 3
    </Box>
  </Flex>
</Flex>
```

## Default props

| Prop name | Type   | Default | Description                         |
| :-------- | :----- | :-----: | :---------------------------------- |
| display   | [CSS display property](https://www.w3schools.com/cssref/pr_class_display.asp) |  `flex`  | sets the `display` property for this component |
