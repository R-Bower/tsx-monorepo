Our React components utilize [styled-system](https://github.com/jxnblk/styled-system)'s "system props" to apply a standard set of props to each component.

Most components get the `COMMON` set of props which give the component access to background, border, color, sizing, space, and transition props.  Refer to the table below for each value.
