# System Props

Our React components utilize [styled-system](https://github.com/jxnblk/styled-system)'s "system props" to apply a standard set of props to each component.

Most components get the `COMMON` set of props which give the component access to color, layout, border, and space props.  These groups correspond to the functions by the same name from **styled-system**. Check the [table of style functions](https://github.com/jxnblk/styled-system/blob/master/docs/table.md#core) for more information.
